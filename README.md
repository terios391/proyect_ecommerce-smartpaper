
# SmartPaper - E-commerce Ligero y Escalable 🛒📝   (version DEMO)

SmartPaper es una plataforma de comercio electrónico diseñada como una solución web ágil, responsiva y de alto rendimiento. Su arquitectura está pensada para potenciar la digitalización de negocios locales, ofreciendo un catálogo de productos intuitivo y un flujo de compra rápido sin los tiempos de carga de plataformas más pesadas.

## 🚀 Valor del Proyecto
* **Carga Optimizada:** Código HTML y CSS puro, sin librerías innecesarias que ralenticen la experiencia del usuario.
* **Diseño Responsivo:** Interfaz fluida basada en CSS Grid y Flexbox, adaptable a cualquier dispositivo móvil o de escritorio.
* **Mantenimiento Eficiente (DRY):** Sistema de inyección de componentes (Encabezado/Footer) mediante JavaScript puro, lo que permite actualizar el catálogo masivo modificando un solo archivo.
* **Gestión Segura:** Sistema de autenticación de usuarios implementando Programación Orientada a Objetos en PHP (Backend en desarrollo).

##  Stack Tecnológico
* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **Backend:** PHP 8+ (POO).
* **Control de Versiones:** Git & GitHub.
